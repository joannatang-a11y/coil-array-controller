# coil-array-controller
Python-based control system and GUI for programmable electromagnetic coil array 
This project implements a Python-based control system for a 4x8 electromagnetic coil array used for ferrofluid manipulation.
The system supports two actuation modes: single-coil mode for rapid transport and double-coil mode for improved stability and precision. Notes: code weregenerated using AI-assisted tools and modified for this project
